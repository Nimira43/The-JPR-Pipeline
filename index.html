The JPR Data Processing Pipeline

Objective

Create a pipeline where JavaScript generates data, Python processes it, and Rust performs some computation before sending it back to JavaScript for final output.

  • JavaScript: Generate random data using seedrandom.js.
  • Python: Read the data, process it, and pass it to Rust.
  • Rust: Perform a computation (e.g., calculating the mean or standard deviation) and pass the results back to Python.
  • JavaScript: Display the final results.
